display manager yang digunakan = GNOME Display Manager
cara mengetahuinya adalah dengan melihat file display-manager.service pada folder /etc/systemd/sytem

window manager yang digunakan = Metacity (Marco)
cara mengetahuinya dengan perintah wmctrl -m pada shell/terminal

desktop environment yang digunakan =  MATE
cara mengetahuinya dengan perintah shell echo $XDG_CURRENT_DESKTOP

shell yang digunakan = bash shell
cara mengetahuinya dengan perintah echo $0
